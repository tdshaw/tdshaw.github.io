HotBrain AI

Created by: Shane Dyrdahl, Abraham Gomez, Chris Mead, Logan Sevcik, and Tucker Shaw

Software Engineering Project

This project was an attempt at creating a dating system for people to connect through common interests. The unique part is that the common interests were detected through a series of videos the users watch while the BrainBit scanner ran an analysis on their brain waves.

We created a Docker container to host our website and database system.

We created a website where our users could register and check their compatible matches.

I created a graphical user interface (GUI) for the users to record their brainwaves while watching videos served by the website.

We created an algorithm that parsed each brain wave file and matched users with specific preferences.

I recorded a video demo for our final if you would like to check it out!
